Fun Side Projects

Connect Four


I wanted to revisit the Connect Four AI and interactive game I had created in Snap! for my first CS class at Berkeley, CS10. I worked hard to learn about minimax AI and to subsequently implement what I had learned in a Python 3 terminal-based game. I had trouble sharing what I had made with friends, so I learned JavaScript in order to rewrite the project for a client-side browser interface. Porting the program from Python to JavaScript was really rewarding because I got to know the subtleties of both languages. Try to win against my AI! Blah blah.
