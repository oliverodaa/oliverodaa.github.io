Many people, whether out shopping, reading the newspaper, or watching a movie come across a challenging problem: how do you find a way for a knight to move around a chess board in such a way that it lands on each square exactly once? This is my JavaScript/HTML/CSS solution of the Knight's Tour Problem.

It works by taking advantage of a heuristic method called Waldorff's rule. This avoids the problem of having to recursively check every possible combination of moves the knight could take. Instead, the code simply analyses the valid moves the knight could make at each point, and chooses the one with the fewest valid moves from it. I hope my code solves your daily struggle to solve this problem. Enjoy!
